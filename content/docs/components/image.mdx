A versatile image component built on Next.js Image and shadcn/ui's AspectRatio. Features three intuitive sizing modes to prevent distortion and work naturally in any layout. Perfect for avatars, product grids, hero banners, and responsive content.

## Sizing Modes

The Image component supports three sizing modes:

1. **Explicit** - Specify both `width` and `height` for fixed dimensions
2. **Constrained** - Specify one dimension + `aspectRatio` (other dimension calculated)
3. **Responsive** - Specify only `aspectRatio` with Tailwind classes for container-controlled sizing

## Error Handling

The component automatically handles image loading errors with fallback content.

```tsx

  src={failedUrl}
  alt="Fallback Demo"
  width={80}
  aspectRatio="1:1"
  rounded="full"
/>

// Custom fallback content
<Image
  src={failedUrl}
  alt="Custom Fallback"
  width={80}
  aspectRatio="1:1"

  alt="Image"

/>
```

## Usage Guidelines

### Choosing Sizing Modes

**Explicit (width + height)**: Use when you need exact dimensions without aspect ratio constraints. Perfect for logos, icons, or images with specific requirements.

```tsx
<Image src={logo} alt="Logo" width={200} height={80} />
```

**Width-Constrained (width + aspectRatio)**: Use when you want a fixed width with proportional height. Ideal for cards, thumbnails, and sidebar images.

```tsx
<Image src={thumb} alt="Thumbnail" width={300} aspectRatio="16:9" />
```

**Height-Constrained (height + aspectRatio)**: Use when you want a fixed height with proportional width. Great for portrait images and story formats.

```tsx
<Image src={story} alt="Story" height={600} aspectRatio="9:16" />
```

**Responsive (aspectRatio + className)**: Use when the container should control sizing. Perfect for grids, flex layouts, and responsive designs.

```tsx
<Image src={hero} alt="Hero" aspectRatio="16:9" className="w-full max-w-6xl" />
```

### Aspect Ratio Guide

- **16:9** - Widescreen (videos, hero banners, landscape photos)
- **9:16** - Portrait/Story (mobile screenshots, vertical content)
- **1:1** - Square (profile images, product thumbnails, Instagram-style posts)
- **4:3** - Standard (traditional photography, presentations)
- **3:2** - Classic photography ratio
- **21:9** - Ultra-wide (cinematic banners, wide hero sections)
- **2:3** - Portrait photography

### Common Pitfalls

**Portrait Aspect Ratios with Fixed Width**

```tsx
// ‚ùå Creates very tall image (192px √ó 341px)
<Image width={192} aspectRatio="9:16" />

// ‚úÖ Use height-constrained instead
<Image height={400} aspectRatio="9:16" />

// ‚úÖ Or use responsive with max-width
<Image aspectRatio="9:16" className="w-full max-w-xs" />
```

**Grid Layouts**

```tsx
// ‚ùå Fixed width breaks grid
<div className="grid grid-cols-3">
  <Image width={300} aspectRatio="1:1" />
</div>

// ‚úÖ Let grid control width
<div className="grid grid-cols-3 gap-4">
  <Image aspectRatio="1:1" className="w-full" />
</div>
```

**Flex Layouts**

```tsx
// ‚ùå Fixed width fights flex
<div className="flex gap-4">
  <Image width={300} aspectRatio="1:1" />
</div>

// ‚úÖ Use flex-1 with min-width
<div className="flex gap-4">
  <Image aspectRatio="1:1" className="flex-1 min-w-0" />
</div>
```

### Accessibility

Always provide descriptive alt text:

```tsx
// ‚úÖ Good
<Image src={product} alt="Red leather backpack with bronze buckles" />

// ‚ùå Bad
<Image src={product} alt="Product image" />

// ‚úÖ Decorative images
<Image src={decoration} alt="" />
```

### Performance

Use `priority` for above-the-fold images:

```tsx
<Image
  src={hero}
  alt="Hero"
  aspectRatio="16:9"
  className="w-full"
  priority
/>
```

Images are lazy-loaded by default for optimal performance.

```tsx
<Image {...props} className="ml-auto" />

// Left (default)
<Image {...props} />
```

## Migration from Old API

If you have existing code using the old API, update as follows:

| Old API | New API |
|---------|---------|
| `size="small"` | `preset="sm"` or `width={96}` |
| `size="medium"` | `preset="md"` or `width={192}` |
| `size="full"` | `className="w-full"` (with aspectRatio) |
| `shape="circular"` | Use `Avatar` component or `rounded="full"` |
| `align="center"` | `className="mx-auto"` |
| `align="right"` | `className="ml-auto"` |
| `fullWidthOnMobile` | `className="w-full sm:w-96"` |
| `fit="fill"` | `fit="cover"` |
| `fit="fit"` | `fit="contain"` |
