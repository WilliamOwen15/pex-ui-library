# Table Collection Component

A comprehensive Table Collection component built on TanStack Table and shadcn/ui that provides sorting, filtering, pagination, row selection, and customizable row actions. Perfect for displaying tabular data with rich interactions.

## Basic Usage

The simplest form displays a table with default styling and no interactive features.

<ComponentPreview
	name="Basic Table"
	code={`import { TableCollection } from "@/components/glide/table-collection";
import { type ColumnDef } from "@tanstack/react-table";

const columns: ColumnDef<User>[] = [
  { accessorKey: "name", header: "Name" },
  { accessorKey: "email", header: "Email" },
  { accessorKey: "role", header: "Role" },
  { accessorKey: "status", header: "Status" },
];

<TableCollection
  data={users}
  columns={columns}
/>`}
>
	<BasicTable />
</ComponentPreview>

## Column Helpers

The component includes helper functions for common column patterns.

### createSelectColumn()

Creates a checkbox column for row selection.

```tsx
import { createSelectColumn } from "@/components/glide/table-helpers";

const columns = [
	createSelectColumn(),
	// ... other columns
];
```

### createSortableColumn(accessorKey, label)

Creates a sortable column with header button and sort indicators.

```tsx
import { createSortableColumn } from "@/components/glide/table-helpers";

const columns = [
	createSortableColumn("name", "Name"),
	createSortableColumn("email", "Email"),
];
```

### createImageColumn(accessorKey, label, nameAccessorKey?)

Creates a column displaying avatars or images with fallback initials.

```tsx
import { createImageColumn } from "@/components/glide/table-helpers";

const columns = [
	createImageColumn("avatar", "User", "name"),
	// Creates avatar with initials from name field
];
```

### createActionsColumn(renderActions)

Creates a column with a dropdown menu for row actions.

```tsx
import { createActionsColumn } from "@/components/glide/table-helpers";
import { DropdownMenuItem } from "@/components/ui/dropdown-menu";

const columns = [
	createActionsColumn((row) => (
		<>
			<DropdownMenuItem onClick={() => edit(row.original)}>
				Edit
			</DropdownMenuItem>
			<DropdownMenuItem onClick={() => remove(row.original)}>
				Delete
			</DropdownMenuItem>
		</>
	)),
];
```

### createBadgeColumn(accessorKey, label, getBadgeVariant?)

Creates a column displaying badges with dynamic variants.

```tsx
import { createBadgeColumn } from "@/components/glide/table-helpers";

const columns = [
	createBadgeColumn("status", "Status", (value) => {
		if (value === "Active") return "default";
		if (value === "Pending") return "secondary";
		if (value === "Inactive") return "destructive";
		return "outline";
	}),
];
```

### createDateColumn(accessorKey, label, formatOptions?)

Creates a column with formatted dates.

```tsx
import { createDateColumn } from "@/components/glide/table-helpers";

const columns = [
	createDateColumn("createdAt", "Created"),
	// Uses default format: Jan 15, 2024

	createDateColumn("updatedAt", "Updated", {
		year: "numeric",
		month: "long",
		day: "numeric",
	}),
	// Custom format: January 15, 2024
];
```

### createCurrencyColumn(accessorKey, label, currency?)

Creates a column displaying currency values.

```tsx
import { createCurrencyColumn } from "@/components/glide/table-helpers";

const columns = [
	createCurrencyColumn("price", "Price"),
	// Displays as $299.99

	createCurrencyColumn("cost", "Cost", "EUR"),
	// Displays as â‚¬299.99
];
```

### createNumberColumn(accessorKey, label, formatOptions?)

Creates a column with formatted numbers.

```tsx
import { createNumberColumn } from "@/components/glide/table-helpers";

const columns = [
	createNumberColumn("quantity", "Qty"),
	// Right-aligned numbers

	createNumberColumn("percentage", "Rate", {
		style: "percent",

```

## Props Reference

### TableCollection

| Prop                  | Type                              | Default       | Description                                     |
| --------------------- | --------------------------------- | ------------- | ----------------------------------------------- |
| `data`                | `TData[]`                         | -             | Array of data to display (required)             |
| `columns`             | `ColumnDef<TData>[]`              | -             | TanStack Table column definitions (required)    |
| `variant`             | `"default" \| "minimal" \| "striped"` | `"default"`   | Visual style variant                            |
| `size`                | `"default" \| "compact"`          | `"default"`   | Table density                                   |
| `limit`               | `number`                          | -             | Limit displayed rows                            |
| `pageSize`            | `number`                          | `10`          | Rows per page when pagination enabled           |
| `searchPlaceholder`   | `string`                          | `"Search..."` | Placeholder text for search input               |
| `emptyMessage`        | `string`                          | `"No results."` | Message shown when table is empty             |
| `enableSorting`       | `boolean`                         | `false`       | Enable column sorting                           |
| `enableFiltering`     | `boolean`                         | `false`       | Enable global search/filter                     |
| `enablePagination`    | `boolean`                         | `false`       | Enable pagination                               |
| `enableRowSelection`  | `boolean`                         | `false`       | Enable row selection with checkboxes            |
| `onRowClick`          | `(row: TData) => void`            | -             | Callback when row is clicked                    |
| `showToolbar`         | `boolean`                         | `true`        | Show toolbar area                               |
| `toolbarActions`      | `ReactNode`                       | -             | Custom actions in toolbar                       |
| `className`           | `string`                          | -             | Additional CSS classes for table container      |
| `classNames`          | `object`                          | -             | CSS classes for specific parts (wrapper, toolbar, table, pagination) |

## Responsive Design

The table automatically handles responsive behavior:

- **Desktop**: Full table layout with all columns visible
- **Tablet**: Horizontal scroll for tables wider than viewport
- **Mobile**: Touch-friendly interactions, horizontal scroll
- **Pagination**: Controls stack vertically on small screens
- **Toolbar**: Search and actions stack vertically on mobile

## Accessibility

The Table Collection component follows accessibility best practices:

- Semantic HTML table elements (`<table>`, `<thead>`, `<tbody>`, `<th>`, `<td>`)
- Proper ARIA labels for interactive elements
- `aria-sort` attributes on sortable columns
- Keyboard navigation support:
  - Tab to navigate between interactive elements
  - Enter/Space to activate buttons and checkboxes
  - Enter/Space on clickable rows (when `onRowClick` is provided)
- Screen reader friendly labels for icon-only buttons
- Focus indicators on all interactive elements
- Selected row state properly communicated

## Performance Tips

For optimal performance with large datasets:

1. **Enable Pagination**: Use `enablePagination={true}` and set reasonable `pageSize`
2. **Memoize Columns**: Define columns outside component or use `useMemo`
3. **Limit Rows**: Use the `limit` prop to cap maximum rows
4. **Debounce Search**: The global filter is debounced by default
5. **Virtual Scrolling**: For 1000+ rows, consider react-virtual (future enhancement)

```tsx
const columns = React.useMemo<ColumnDef<User>[]>(
	() => [
		{ accessorKey: "name", header: "Name" },
		{ accessorKey: "email", header: "Email" },
	],
	[],
);
```

## Common Patterns

### Master-Detail View

```tsx
<TableCollection
	data={items}
	columns={columns}
	onRowClick={(row) => setSelectedItem(row)}
/>
{selectedItem && <DetailPanel item={selectedItem} />}
```

### Bulk Actions

```tsx
const [selection, setSelection] = useState({});

<TableCollection
	data={items}
	columns={columns}
	enableRowSelection={true}
	toolbarActions={
		Object.keys(selection).length > 0 && (
			<>
				<Button onClick={() => bulkEdit(selection)}>Edit Selected</Button>
				<Button variant="destructive" onClick={() => bulkDelete(selection)}>
					Delete Selected
				</Button>
			</>
		)
	}
/>
```

### External State Control

```tsx
const [sorting, setSorting] = useState<SortingState>([]);

// Use TanStack Table directly for full control
const table = useReactTable({
	data,
	columns,
	state: { sorting },
	onSortingChange: setSorting,
	// ... other options
});
```
